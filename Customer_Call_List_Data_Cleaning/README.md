# Call List Data Cleaning

## Objective
Clean and standardize a raw call list dataset containing phone numbers and addresses.

## Key Cleaning Steps
- Removed non-numeric characters from phone numbers
- Standardized phone number format (XXX-XXX-XXXX)
- Extracted extra address information
- Handled missing and invalid values

## Tools Used
- Python
- Pandas
- NumPy

## Output
Clean dataset saved to `data/clean/call_list_clean.csv`

## Notes
This project focuses on data cleaning and validation, not modeling.
